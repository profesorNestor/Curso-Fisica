Vectores Unitarios: Los Guardianes del Plano 🛡️📐⚖️

🛡️

¿Qué son los Vectores Unitarios?

Los vectores unitarios son vectores especiales que tienen magnitud igual a 1 y señalan las direcciones fundamentales del espacio.

î

Vector Unitario î (i-hat)

📍 Dirección: Eje X positivo (→)

📏 Magnitud: |î| = 1

📐 Coordenadas: î = (1, 0)

🎯 Función: Define la dirección horizontal hacia la derecha

ĵ

Vector Unitario ĵ (j-hat)

📍 Dirección: Eje Y positivo (↑)

📏 Magnitud: |ĵ| = 1

📐 Coordenadas: ĵ = (0, 1)

🎯 Función: Define la dirección vertical hacia arriba

Propiedades Matemáticas Fundamentales

📏 Magnitud Unitaria

|î| = |ĵ| = 1

⟂ Perpendiculares

î ⊥ ĵ

🔄 Producto Punto

î · ĵ = 0

🧮 Representación General de un Vector

$\vec{v} = v_x \hat{i} + v_y \hat{j}$

Donde $v_x$ y $v_y$ son las componentes escalares del vector

Componentes Rectangulares 📊📐⚖️

¿Qué son las Componentes Rectangulares?

Las componentes rectangulares son las "proyecciones" de un vector sobre los ejes coordenados perpendiculares.

Descomposición Trigonométrica

Dado un vector $\vec{v}$ con magnitud |$\vec{v}$| y ángulo θ respecto al eje X positivo:

🧮 Fórmulas Fundamentales:

Componente X: $v_x = |\vec{v}| \cos θ$

Componente Y: $v_y = |\vec{v}| \sin θ$

Vector completo: $\vec{v} = v_x\hat{i} + v_y\hat{j}$

Descomposición en el Plano XY

Vector $\vec{v}$ descompuesto en sus componentes $v_x$ y $v_y$

Propiedades Matemáticas Clave

📐 Magnitud del Vector

$|\vec{v}| = \sqrt{v_x^2 + v_y^2}$

Teorema de Pitágoras aplicado

🧭 Ángulo de Dirección

$θ = \arctan\left(\frac{v_y}{v_x}\right)$

Función tangente inversa

🔄 Componentes Independientes

$v_x \perp v_y$

Ortogonales entre sí

🔄 Reconstrucción del Vector Original

$\vec{v} = v_x\hat{i} + v_y\hat{j} = |\vec{v}|\cos θ\hat{i} + |\vec{v}|\sin θ\hat{j}$

Cualquier vector puede expresarse como la suma de sus componentes perpendiculares

Ejemplo Práctico: Fuerza sobre un Bloque

Una fuerza de 100 N se aplica sobre un bloque formando un ángulo de 37° con la horizontal:

📊 Cálculo de Componentes:

Datos:

• |F⃗| = 100 N

• θ = 37°

Componentes:

• F_x = 100 cos(37°) = 100 × 0.8 = 80 N

• F_y = 100 sin(37°) = 100 × 0.6 = 60 N

📦

Bloque con Fuerza Aplicada

Representación vectorial:
$\vec{F} = 80\hat{i} + 60\hat{j}$ N

Verificación:
$|\vec{F}| = \sqrt{80^2 + 60^2} = \sqrt{6400 + 3600} = \sqrt{10000} = 100$ N ✓

Ejemplo Práctico: Navegación de un Drone 🚁💡

📝 Problema:

Un drone se mueve 200 m hacia el este y luego 150 m hacia el norte. ¿Cuál es su desplazamiento total?

📊 Datos del problema:
  • Vector A: $\vec{A} = 200\,\hat{i}$ (este)
  • Vector B: $\vec{B} = 150\,\hat{j}$ (norte)
  • Encontrar: $\vec{R} = \vec{A} + \vec{B}$
🔍 Solución por Método del Triángulo:
Paso 1: Traza $\vec{A}$ hacia el este.
Paso 2: Desde la punta de $\vec{A}$, traza $\vec{B}$ hacia el norte.
Paso 3: $\vec{R}$ es el vector desde el origen al punto final.
$\boxed{\lvert\vec{R}\rvert = \sqrt{200^2 + 150^2} = 250\,\mathrm{m}}$
$\theta = \arctan\left(\frac{150}{200}\right) = 36.87^\circ$
$\vec{R} = 200\,\hat{i} + 150\,\hat{j}$

Construcción Gráfica

Triángulo rectángulo formado por los vectores A, B y R

💡 Interpretación:
El drone terminó a 250 metros del origen, en una dirección 36.87° al noreste.
¡Ese es su desplazamiento neto! 🚁✨

Construcción Paso a Paso ▱📐✨

Paso 1-2 🟠

Vectores desde el origen

Paso 3 🟢

Líneas paralelas

Paso 4 🟣

Vector resultante (diagonal)

📏 Ley del Paralelogramo

$\vec{R} = \vec{A} + \vec{B}$

La diagonal del paralelogramo representa la suma vectorial

💡 Dato curioso: Este método fue usado por primera vez por Pierre-Simon Laplace en 1799 para estudiar fuerzas mecánicas.

Ejemplo Práctico: Fuerzas en una Grúa 🏗️⚡📐

🏗️ Problema:

En un punto de una grúa actúan dos fuerzas:
F₁ = 80 N a 30° y F₂ = 60 N a 120° (medidos desde el eje X positivo).
Encuentra la fuerza resultante.

📊 Datos del problema:
  • Fuerza 1: $\vec{F}_1 = 80\ \mathrm{N}$ a 30°
  • Fuerza 2: $\vec{F}_2 = 60\ \mathrm{N}$ a 120°
  • Encontrar: $\vec{F}_R = \vec{F}_1 + \vec{F}_2$
  • Método: Paralelogramo

🔍 Solución por Paralelogramo:

🟠 Paso 1: Componentes de F₁

$F_{1x} = 80\cos(30^\circ) = 69.3\ \mathrm{N}$

$F_{1y} = 80\sin(30^\circ) = 40.0\ \mathrm{N}$

🔵 Paso 2: Componentes de F₂

$F_{2x} = 60\cos(120^\circ) = -30.0\ \mathrm{N}$

$F_{2y} = 60\sin(120^\circ) = 52.0\ \mathrm{N}$

🟣 Paso 3: Fuerza Resultante

$F_{Rx} = F_{1x} + F_{2x} = 39.3\ \mathrm{N}$

$F_{Ry} = F_{1y} + F_{2y} = 92.0\ \mathrm{N}$

📏 Resultado Final:

🔸 Magnitud: $|\vec{F}_R| = \sqrt{(39.3)^2 + (92.0)^2} = \color{#eab308}{100\ \mathrm{N}}$

🔸 Dirección: $\theta = \arctan\left(\frac{92.0}{39.3}\right) = \color{#059669}{66.9^\circ}$

🔸 Vector: $\vec{F}_R = \color{#8b5cf6}{39.3\,\hat{i} + 92.0\,\hat{j}}$ N

▱ Construcción del Paralelogramo

Paralelogramo de fuerzas con diagonal resultante.
F₁, F₂ y $\vec{F}_R$.

✅ Verificación:

El paralelogramo muestra que la diagonal (vector naranja) representa correctamente la suma de las dos fuerzas.
La magnitud y dirección calculadas coinciden con la construcción geométrica.

🏗️ Interpretación física: La grúa experimenta una fuerza neta de 100 N en dirección 66.9° respecto a la horizontal.

📒 Fórmulas Fundamentales

🚩 Suma de Componentes:

\( R_x = A_x + B_x + C_x + \ldots \)
\( R_y = A_y + B_y + C_y + \ldots \)
🔹 Rx y Ry son la suma de todas las componentes en X y Y, respectivamente.

🌟 Vector Resultante:

\( |\vec{R}| = \sqrt{R_x^2 + R_y^2} \)
\( \theta = \arctan\left(\frac{R_y}{R_x}\right) \)
🔹 \(|\vec{R}|\) es la magnitud total y θ es el ángulo del vector respecto al eje X.
✏️ Recuerda: Primero suma componentes, después calcula módulo y ángulo usando las fórmulas del resultado.

Ejemplo Práctico: Sistema de Fuerzas en un Puente 🧮⚙️🌉

Planteamiento del Problema

🌉 En el diseño de un puente, tres fuerzas actúan sobre un nodo crítico:
  • Fuerza A: 150 N a 25° sobre la horizontal
  • Fuerza B: 200 N a 135° sobre la horizontal
  • Fuerza C: 120 N a 270° (hacia abajo)
🔎 Encontrar: La fuerza resultante del sistema usando el método de componentes.

Datos Organizados del Sistema

🔗 Vector ⚡ Magnitud 📐 Ángulo cos θ sin θ
Fuerza A 150 N 25° 0.906 0.423
Fuerza B 200 N 135° -0.707 0.707
Fuerza C 120 N 270° 0.000 -1.000
¡Tabla útil para organizar y visualizar los cálculos!
📊 Suma de Componentes X
Aₓ = 150 × cos(25°) = 135.9 N
Bₓ = 200 × cos(135°) = -141.4 N
Cₓ = 120 × cos(270°) = 0.0 N
ΣFₓ = 135.9 + (−141.4) + 0.0 = −5.5 N
📊 Suma de Componentes Y
Aᵧ = 150 × sin(25°) = 63.5 N
Bᵧ = 200 × sin(135°) = 141.4 N
Cᵧ = 120 × sin(270°) = -120.0 N
ΣFᵧ = 63.5 + 141.4 + (−120.0) = 84.9 N
🧮 Vector Resultante:
R⃗ = ΣFₓ î + ΣFᵧ ĵ
R⃗ = −5.5î + 84.9ĵ N
📏 Magnitud del Vector Resultante
|R⃗| = √((-5.5)² + (84.9)²) = 85.1 N
🧭 Dirección del Vector Resultante
θ = arctan(84.9 / −5.5) = −86.3°
θ = 93.7° desde el eje X
Como Rx < 0 y Ry > 0, el vector está en el II cuadrante.
Ángulo desde eje X positivo: 180° − 86.3° = 93.7°

Resultado Final

R⃗ = −5.5î + 84.9ĵ N
|R⃗| = 85.1 N   |   θ = 93.7°
🌉 Interpretación para Ingeniería:

El nodo del puente experimenta una fuerza neta de 85.1 N, en dirección 93.7° (casi vertical ascendente).
¡Clave para garantizar la estabilidad y seguridad estructural!

¡Recuerda! Primero descompón cada vector en componentes, suma X e Y por separado, y luego aplica Pitágoras y trigonometría para obtener el resultado final.
¡Organiza tu trabajo en tablas y bloques de colores para evitar errores!

Vector Negativo: El Espejo Matemático ⇄🔄⚡

¿Qué es un Vector Negativo?

El vector negativo de \( \vec{A} \), denotado como \( -\vec{A} \), es un vector con igual magnitud pero sentido opuesto.
¡Matemáticamente, es como un espejo perfecto en la dirección contraria!

Comparación visual entre \( \vec{A} \) y \( -\vec{A} \): misma longitud, direcciones opuestas.
📏

Magnitud Idéntica

\( |\vec{A}| = |-\vec{A}| \)
La longitud del vector no cambia, solo su orientación.

🔄

Sentido Opuesto

Si \( \vec{A} \) apunta hacia el noreste,
\( -\vec{A} \) apunta hacia el suroeste.

⭕

Suma es Vector Cero

\( \vec{A} + (-\vec{A}) = \vec{0} \)
Se cancelan mutuamente, dando el vector nulo.

Relaciones Matemáticas Fundamentales

Propiedades Básicas:

\( |-\vec{A}| = |\vec{A}| \)

\( \vec{A} + (-\vec{A}) = \vec{0} \)

\( -(-\vec{A}) = \vec{A} \)

En Componentes:

Si \( \vec{A} = A_x\hat{i} + A_y\hat{j} \)

Entonces \( -\vec{A} = -A_x\hat{i} - A_y\hat{j} \)

Resta de Vectores: Suma Disfrazada ➖🎭⚡

El Secreto de la Resta Vectorial

¡No existe una "resta" especial de vectores!
La operación $\vec{A} - \vec{B}$ es simplemente $\vec{A} + (-\vec{B})$.
¡Es una suma disfrazada! 🎭✨

Visualización: $\vec{A} - \vec{B}$ es $\vec{A} + (-\vec{B})$.
Suma gráfica y analítica de vectores con su negativo.

La Transformación Mágica

Problema Original

$\vec{A} - \vec{B}$
¿Cómo resto vectores?
🤔 Parece complicado...
⟶
¡Transformación!

Problema Transformado

$\vec{A} + (-\vec{B})$
¡Ahora es una suma!
🎉 ¡Ya sabemos sumar!

Métodos Visuales y Analíticos para Restar Vectores

Método del Triángulo

1. Invierte $\vec{B}$ para obtener $-\vec{B}$
2. Conecta la cola de $-\vec{B}$ con la cabeza de $\vec{A}$
3. El resultado va del origen a la punta final
Ideal para visualizar gráficamente

Método del Paralelogramo

1. Coloca $\vec{A}$ y $-\vec{B}$ desde el origen
2. Completa el paralelogramo
3. La diagonal es $\vec{A} - \vec{B}$
Elegante y simétrico

Método por Componentes

1. $(A - B)_x = A_x - B_x$
2. $(A - B)_y = A_y - B_y$
3. Reconstruye el vector resultante
¡Máxima precisión analítica!

🎯 Fórmula Unificada de la Resta Vectorial

$\vec{A} - \vec{B} = \vec{A} + (-\vec{B})$
Recuerda: La resta de vectores es la suma del vector negativo.

Ejemplo Visual: Resta de Vectores Perpendiculares 🟦➖🟩

🚴‍♂️ Problema: Ciclista Vectorial

Un ciclista avanza 4 metros al Este ($\vec{A}$) y luego 3 metros al Norte ($\vec{B}$).
¿Cuál es el vector $\vec{A} - \vec{B}$? Usa componentes, calcula magnitud y dirección.

Paso a Paso

1️⃣ Descompón en componentes
$\vec{A} = (4,\,0)$
$\vec{B} = (0,\,3)$
(X, Y)
2️⃣ Resta por componentes
$\vec{A} - \vec{B} = (4-0,\,0-3) = (4,\,-3)$
X = 4    Y = -3
3️⃣ Magnitud y dirección
$\left|\vec{A} - \vec{B}\right| = \sqrt{4^2 + (-3)^2} = \sqrt{16+9} = \sqrt{25} = 5$ m
$\theta = \arctan\left(\dfrac{-3}{4}\right) = -36.87^\circ$
(Respecto al eje X, sentido horario: sureste)
Visualización geométrica: $\vec{A} - \vec{B}$
Conclusión:
Al restar vectores perpendiculares, el resultado es un vector oblicuo.
